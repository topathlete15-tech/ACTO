# MATO（マト）

## 概要
MATO（Mission Action TOol）は、プロジェクトやタスクを「目標（Mission）→ 行動（Action）→ツール（TOol）」の流れで、シンプルかつ直感的に管理できるタスク管理システムです。
オレンジを基調としたデザインで、ユーザーが「何をすべきか」をすぐに把握でき、効率的にタスクを進められることを目的としています。

## サイトテーマ
「目標を定め、アクションに落とし込む。MATOとともに。」

## テーマを選んだ理由
現場やプロジェクトにおいて、以下のような課題をよく見かけます。
- タスクが散乱して、何を優先すべきか曖昧になる  
- 目標は立てても、日々の行動に落とし込まれていない  
- 進捗が“見える化”されておらず、個人の行動の動きが鈍くなる
- 当然、このような状況ではメンバーのモチベーションも上がらず果たして効率的で効果的なタスクはできるのでしょうか？

これらを解決するために、MATOでは「目標を明確に → 行動に変換 → タスクを管理・完了へ」というシンプルな流れを設計しました。  
この流れが日々の“進め方”そのものを変えることができると考えています。

### きっかけ
「目標を立てたのに振り返る時間がなくて置き去りにされた」「やることを書き出したのに動けなかった」そんな経験から、  
タスクを“設定して終わり”にせず、「行動と結果に直結させる」ツールが必要だと感じました。  
MATOはその“抜け”を埋めるためのプロダクトです。

### 問題点
- プロジェクトの目標と個々のタスクが乖離している  
- タスク数が多すぎて優先順位がつけられない  
- 自分が何をしなくてはいけないのか可視化できる仕組みがない  

### 解決策 
- タスク進捗をチェックボックス／ステータスで可視化   
- シンプルなUI／操作性を重視（アイコン：オレンジ基調、ミニマルデザイン）  

## ターゲットユーザー
- 個人でタスクを抱えている人（社内・個人問わず）  
- “やるべきこと”を整理し、日々着実に進めたい人  

## 主な利用シーン 
- 日常タスク：個人のToDoを日々管理、進捗を可視化  

## 機能一覧  
- タスク（Action）登録／編集／削除／ステータス変更（未着手／進行中／完了）  
- タスクに対するコメント機能      
- レスポンシブ対応（PC／スマホ）  

## 開発環境
- OS：macOS／Windows／Linux いずれも対応  
- 言語・フレームワーク：現在未定
- バージョン管理：Git + GitHub
