
# ACTO（アクト）

## サービス概要

**「思考を止めて、行動を開始する」**

ACTOは、個人の実行力を最大化するためのタスク実行ツールです。
複雑な目標設定や計画作りは一切排除。「やるべきこと」を書き出し、オレンジを基調とした直感的なUIで「未着手・進行中・完了」のステータスを切り替えていくことだけに集中させます。

### サイトテーマ

> **「迷いを捨てて、動き出す。シンプル・タスク・アクション。」**

## 開発背景

高機能なタスク管理ツールを使っても、以下のような経験はないでしょうか？

* 計画を立てただけで満足してしまう
* 機能が多すぎて管理そのものが続かない
* 「あれもやらなきゃ」と頭の中で考えてフリーズしてしまう

ACTOは、これらの原因となる「目標（Mission）」の管理機能をあえて排除し、**「行動（Action）」**のみにフォーカスしました。
脳内のタスクを全てツールに預け、ユーザーは目の前の作業を消化することだけにエネルギーを使えるように設計されています。

### 本アプリのこだわり

* **No Social**
* 他人の目を気にする必要はありません。自分だけの空間でタスクに向き合えます。


* **No Planning**
* 未来の壮大な計画よりも、「今日、今、何をするか」を最優先します。



## 主な利用シーン

* 日常のToDo管理（買い物、振込、メール返信など）
* 学習の進捗管理（「Rubyの章を進める」「Qiitaを読む」など）
* 備忘録としてのタスク登録

## 機能一覧

### 1. アクション管理（Core Features）

| 機能 | 概要 |
| --- | --- |
| **直感的なタスク操作** | 登録・編集・削除をストレスなく行えます。 |
| **3段階ステータス** | **未着手 (Pending)**: とりあえず書き出した状態<br>

<br>**進行中 (In Progress)**: 今まさに取り組んでいる状態<br>

<br>**完了 (Completed)**: 終わらせた達成感を味わう状態 |
| **カテゴリ分け** | 散らばるタスクをジャンルごとに整理できます。 |

### 2. 行動ログ・振り返り（Self Feedback）

| 機能 | 概要 |
| --- | --- |
| **タスクメモ** | 作業中の気づきや、次回への申し送り事項を記録。自分専用のメモ帳として活用します。 |
| **重要マーク** | 特に優先すべきタスクや、完了して嬉しかったタスクに視覚的なアクセントを付与します。 |
